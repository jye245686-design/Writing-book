# AI 小说生成 - 前端

简约扁平风格的 Web 端，基于 React + Vite + Tailwind CSS。

## 开发

```bash
cd web
npm install
npm run dev
```

浏览器访问终端提示的本地地址（通常为 http://localhost:5173）。

## 构建

```bash
npm run build
```

## 设计说明

- **简约**：留白充足、层级清晰、仅保留必要元素。
- **扁平**：无渐变、无重阴影，以纯色与 1px 边框为主；按钮与卡片使用 `card-flat`、`btn-flat`、`input-flat` 等类。
